# Project Analysis - MiniVault

## 6a) Strategic objectives

**What strategic objectives of the company does your software address?**  
My software helps improve data security and privacy by allowing users to store passwords safely. It also supports digital transformation by replacing insecure methods like writing passwords on paper or reusing them.

**How does the software align with the company's digital strategy?**  
It aligns by digitizing credential management and improving security practices, which is a key part of any digital strategy.

---

## 6b) Business and communication areas

**What areas of the company benefit the most from your software?**  
Mainly the business and IT areas, because they deal with accounts, tools and access to services. Communication can also benefit by securely managing shared accounts.

**What operational impact do you expect in daily activities?**  
It makes daily work easier and safer, reducing time spent recovering passwords and avoiding security issues.

---

## 6c) Areas susceptible to digitalization

**What areas can be digitalized with your software?**  
Credential management, access control and account organization.

**How does digitalization improve these areas?**  
It improves organization, security and efficiency, since everything is stored in one place and protected.

---

## 6d) Integration of digitalized areas

**How do digitalized areas interact with non-digitalized ones?**  
Digitalized areas replace manual processes, but still depend on users interacting with the system.

**What improvements would you propose to integrate them better?**  
Training users and creating clear processes so everyone uses the software correctly.

---

## 6e) Present and future needs

**What current needs does your software solve?**  
It solves the need to store and manage passwords securely.

**What future improvements could be implemented?**  
Cloud synchronization, multi-device access and stronger encryption systems.

---

## 6f) Relation with technologies

**What technologies have you used and how do they impact the company?**  
I used web technologies like HTML, CSS and JavaScript. They allow the app to run in any browser without installation.

**What benefits do these technologies provide?**  
They make the software accessible, easy to use and compatible with many devices.

---

## 6g) Security gaps

**What possible security risks exist in your software?**  
Data could be lost if the user clears browser storage, and localStorage is not as secure as a fully encrypted system.

**What measures would you propose to reduce these risks?**  
Implement stronger encryption, backups and possibly cloud storage with authentication.

---

## 6h) Data management and analysis

**How is data managed in your software?**  
Data is stored locally in the browser using localStorage.

**How do you ensure data quality and consistency?**  
By controlling how data is saved and updated within the application.
