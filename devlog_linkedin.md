# 🚀 Devlog: Building MiniVault - My First Open-Source Password Manager

I’m thrilled to share a project I’ve been working on: **MiniVault**! 🔐

As a student project, the goal was simple: create a tool that is **realistic, clean, and secure**. We often rely on cloud-based password managers, but I wanted to build something that puts the user back in full control of their data.

### 🛠️ The Stack:
- **Python** & **PySide6**: For a native, high-performance desktop experience.
- **Cryptography**: Implementing industry-standard AES-256 (Fernet) and PBKDF2 for key derivation.
- **Streamlit**: To create a quick, accessible web demo for the core features.

### 💡 What I learned:
- **Zero-Knowledge Security**: Designing a system where the app never "knows" your password, only the derived key.
- **Modern UI/UX**: Moving beyond basic styles to create a sleek dark theme that feels premium.
- **Open Source Ready**: Preparing a repository with proper documentation, contribution guides, and licensing.

Check out the repo and the web demo! Looking forward to hearing your thoughts.
